# ProjectX Server

Compose-first application factory server implementation.

## Quick Start

```bash
# Install dependencies
bun install

# Generate documentation
bun run docs:build

# Start development server
bun run dev
```

## Documentation

After running `bun run docs:build`, open:
```
docs/generated/index.html
```
